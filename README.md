# ⏰ 실시간 공동구매 플랫폼 
## 📑 Introduction

Real-Time Group Buy는 사용자들이 특정 제품에 대해 공동구매 그룹을 생성하고, 일정 수의 참여자를 모집하여 할인된 가격에 제품을 구매할 수 있는 실시간 공동구매 플랫폼입니다. <br/>
공동구매의 진행 상황을 실시간으로 관리하고, 제품 재고 및 주문 상태를 동시성 이슈 없이 처리할 수 있도록 설계되었습니다.
